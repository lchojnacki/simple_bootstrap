Czym jest Python?

Python to interpretowany, interaktywny, skryptowy język programowania. posiada w pełni dynamiczny system typów i automatyczne zarządzanie pamięcią, jest zatem podobny do takich języków, jak Tcl, Perl, Scheme czy Ruby. Python realizuje jednocześnie kilka paradygmatów. Podobnie do C++ nie wymusza jednego stylu programowania, pozwalając na stosowanie różnych. Python posiada:

  • Paradygmat programowanie obiektowego
  • Paradygmat programowanie strukturalnego
  • Paradygmat programowanie funkcyjnego

Typy sprawdzane są dynamicznie, a do zarządzania pamięcią stosuje się architekturę garbage collection podobnie jak w języku Ruby czy też D.

Dlaczego programowanie w Pythonie jest takie fajne?

  • Dynamiczne typy danych
  • Niewielka liczba słów kluczowych i operatorów
  • Pełna obiektowość języka
  • Czytelny i schludny kod programu poprzez wymuszanie wcięć
  • Mniejsza objętość kodu, niż w innych językach, np. C++
