  • Pierwsze kroki

    1991

    Guido van Rossum w Centrum Matematyki i Informatyki w Amsterdamie publikuje wersję 0.9.0 języka Python. Jego nazwa pochodzi nie od zwierzęcia, jak jest to powszechnie uważane, lecz od "Latającego cyrku Monty Pythona", którego van Rossum był fanem.

    Guido van Rossum

    2006, From Wikimedia Commons, the free media repository

    Na tym etapie rozwoju języka dostępne były klasy wraz z dziedziczeniem, obsługa wyjątków, funkcje i podstawowe typy danych, takie jak list, dict, str i inne.

  • Wersja 1

    1994

    W styczniu 1994 roku opublikowano wersję 1.0 języka. Głównymi nowościami wprowadzonymi w tym wydaniu były narzędzia takie jak lambda, map, filter czy reduce.

    Ostatnią wersją wydaną przez Centrum Matematyki i Informatyki w Amsterdamie był Python 1.2. Od 1995 roku Van Rossum kontynuował pracę nad Pythonem w Corporation for National Research Initiatives (CNRI) w Reston w Wirginii, gdzie wydał kilka wersji Pythona, do 1.6 włącznie.

  • Wersja 2

    2000

    W 2000 roku van Rossum i zespół pracujący nad rozwojem jądra Pythona przenieśli się do BeOpen.com by założyć zespół BeOpen PythonLabs. Pierwszą i jedyną wersją wydaną przez BeOpen.com był Python 2.0.

    W tej wersji do języka został wprowadzony mechanizm garbage collection, odpowiedzialny za automatyczne zwalnianie nieużywanej pamięci.

    Od wersji 2.1 (2001) język jest rozwijany jako projekt Open Source przez niedochodową organizację Python Software Foundation (PSF), wzorowaną na Apache Software Foundation.

    Python 2.2 uszeregował typy danych i klasy w jednej, zgodnej hierarchii. Wprowadził również mechanizm generatorów, pozwalający oszczędzać pamięć.

    Wprowadzony we wrześniu 2006 Python 2.5 dołączył do języka instrukcję with.

  • Rozwidlenie dróg

    2008

    Od 2008 istnieją dwie gałęzie: Python 2 i Python 3 (zerwanie kompatybilności wstecz). Ostatnią wersją Pythona 2.x (stan na 21.06.2018) jest Python 2.7.13. Python 2.8 nie będzie nigdy wydany, a wsparcie dla wersji 2.7 będzie utrzymywane do 2020 roku. Z tego powodu zalecane jest przejście na Pythona w wersji 3.

  • Python 3

    2008

    W grudniu 2008 został zaprezentowany Python 3 (nazywany również "Python 3000" lub "Py3K"). Wprowadził on duże zmiany, które uniemożliwiły zachowanie kompatybilności z Pythonem 2.

    Najważniejsze nowości wprowadzone w Python 3:

    • polecenie print zastąpiono wbudowaną funkcją print()
    • usunięcie funkcji input() z Pythona 2 i przemianowanie funkcji raw_input() pobierającą string na input()
    • wprowadzenie Unicode jako standardowego kodowania stringów
    • zmiana działania znaku /: w Pythonie 2 5 / 2 zwracało 2, w Pythonie 3 5 / 2 zwraca 2.5; wprowadzono operator dzielenia całkowitego (5 // 2 = 2)

    Ostatnią wydaną wersją jest Python 3.6.1 (21.03.2017).
